POS PWA Warung Makan
====================

Cara pakai:
1. Buka index.html di browser (Chrome/Edge).
2. Tambahkan ke Home Screen untuk PWA offline.
3. Gunakan tab Kasir untuk transaksi, tab Laporan untuk laporan.
4. Klik 'Simpan & Cetak Struk' untuk mencetak struk 58mm.
5. Ubah print.css ke 80mm jika menggunakan printer 80mm.
